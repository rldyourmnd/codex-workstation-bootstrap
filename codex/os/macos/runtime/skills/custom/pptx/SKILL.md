# PPTX (Codex)

## Scope
- Generate new slide decks.
- Edit existing decks safely.
- Extract and summarize slide content.
- Apply layout, template, and structure updates.

## Workflow
1. Confirm audience and objective of the deck.
2. Establish slide narrative before visual polish.
3. Keep one core message per slide.
4. Preserve readability: hierarchy, whitespace, consistent typography.
5. Validate final deck opens correctly and structure is intact.

## Quality Bar
- Clear storyline.
- Consistent visual language.
- No overloaded slides.
