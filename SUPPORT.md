# Support

## Where to ask for help

- Usage and setup questions: open a GitHub Discussion or Issue.
- Bugs and regressions: open an Issue with reproduction details.
- Security concerns: see `SECURITY.md`.

## What to include

- OS and version (for now, macOS target).
- Command used and full output.
- Expected behavior vs actual behavior.
- Relevant file paths and config snippets (with secrets redacted).
