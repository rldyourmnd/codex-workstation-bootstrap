# AGENTS.md Quality Criteria (Codex)

## Scoring Rubric
1. Command Accuracy (20)
2. Workflow Clarity (20)
3. Safety Guardrails (15)
4. Conciseness (15)
5. Currency (15)
6. Actionability (15)

## Core Checks
- Commands map to real scripts/tools in repo.
- Instructions align with current CI/test/lint flow.
- High-risk actions are constrained.
- No contradictory rules.
- No product-specific assumptions unrelated to Codex.
